Customizing the theme

Using the Grub Menu ››

Customize Theme


The background image shown here is from the  "Beachsunset" theme.




This screen will appear when you click the Customize Theme button on the Main Configuration  screen.



               

Click the image to select your background

Click the screenshot image to choose the background to be displayed at boot time.

The background selection screen will then appear.


If you want the Grub boot time console to use text mode instead of a graphics background, select

** No Theme - Text Only **


This dialog lets you add your own background image files. The files must be in JPG format. Max image file size is 4 MB.

See the  Other User Files  topic for details.


You can also select any user background files you want to remove.



Center Menus

Check this to vertically center the menu entries on the boot screen.

You may need to adjust the Grub Graphics Mode on the Main Configuration Screen for centering to work properly.



Set Text Colors

By clicking these buttons, you can set the colors for the following items:


Titles                    - The color of the boot menu titles.


Selected Item       - The color of the boot menu item currently selected.


Text                     - The color of the command prompt, version and boot mode text.


Clock                   - The color of the circular clock ticks and countdown seconds digits.



Enable Grub Timeout

Check this to enable the Grub countdown clock. You can then set the number of seconds for the Grub default timeout.



Style

Select whether you would like a Clock or a Progress Bar to show the time remaining before boot.



Face

Select clock face image to show within the circular timeout clock ticks. This option is only available when you select the clock timeout style above.


Note: You can add our own clock face images. See the  Other User Files  topic for details.



Dark Background

Check this to include a dark background. This will improve the contrast with lighter theme images.



Highlight Selected Item

Check this to highlight the selected menu item for better visibility.



Show Scroll Bar

A scroll bar will appear at boot time if there are enough items to overflow the page.

This can be useful when you have many boot menu items.



Show Grub2Win Version

Check this if you wish the version to appear at boot time.



Show Prompt Lines

Check this if you wish the command prompt help text to appear at boot time.



Show Boot Mode

Check this to show the current boot mode of the machine, either EFI or BIOS. Also shows the architecture of the machine's firmware, either 64 Bit or 32 Bit.



Set Standard View

Resets the theme image options to factory settings.




When you are satisfied with your choices. Click the OK button.
