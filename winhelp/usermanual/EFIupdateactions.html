EFI Update Actions


(EFI mode only)





This screen will appear during EFI setup processing after you click the EFI Partition Operations button on the Main Configuration screen.


It allows you to browse, edit and update files and directories in your EFI partitions.



Browse

Opens a dialog to view and edit files in the EFI partition.


Select a file to edit with Notepad.


You can also use the right mouse button to rename, copy, cut paste and delete files and directories.


Note: Be very careful. It is possible to render your system unbootable if you delete or alter a critical EFI file. Make sure you have good backups of your EFI partitions.




Action

Select an action you would like to perform on your EFI partition.


Actions include the installation, deletion and refresh of the Grub2Win EFI modules.


There are also actions to backup and restore all files and folders in your EFI partition.

The default folder for EFI backups is C:\grub2\windata\backup\efi.partitions.






Once you have selected an action, click the Apply Actions button to start the selected EFI update task.


When you have completed the actions you wish to apply, click the Close button to return to the EFI update log screen.


There you can choose to continue using Grub2Win or exit the program completely.
