EFI Update Log


(EFI mode only)





This screen will appear when you click the Setup EFI Partition Modules button on the Main Configuration screen. It shows the progress of the EFI module install utility.

When the EFI partition has been readied for update, the EFI Update Confirmation screen will appear.


Once you have confirmed the update, processing will go on to completion.


When the setup successfully completes, you can either return to the  Main Configuration  screen or exit the program.
