Rescue Mode

  (EFI systems only)



There are several things that can prevent the normal Grub2Win menu from appearing at boot time:



Syntax errors in the C:\grub2\grub.cfg file.



Missing C:\grub2 directories or corruption in other .cfg files or submenus.



Errors in your graphics card initialization.




If the Grub2Win menu does not come up, you will probably see a black screen with a Grub> command prompt.


Simply enter rescue at the command prompt and a very basic Grub menu will be loaded.


This basic menu will allow you to boot Windows in almost any situation.
