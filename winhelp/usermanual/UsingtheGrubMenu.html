Using the Grub Menu




To configure Grub2Win, either click on the Grub2Win desktop shortcut or run C:\grub2\grub2win.exe. This requires Windows administrator access.


When you run the program, several things will happen:


The Main Configuration screen will appear. This allows you to customize Grub with the OS selections and parameters you want.


Once you are satisfied with your customization, click OK, Grub2Win will set up the main Grub control file,

C:\grub2\grub2.cfg 


If your machine is running in EFI mode, the program will then make additions to the EFI partition to run the Grub2Win menu at boot time.


In BIOS mode, the Windows BCD will be updated to run the Grub2Win menu at boot time.
