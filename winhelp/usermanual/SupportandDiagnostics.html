                     Support and Diagnostics





Grub2Win is maintained at SourceForge. The URL is http://sourceforge.net/projects/grub2win/


There is an active board at SourceForge where many topics related to Grub2Win are discussed.



Please provide a description of the problem as well as screen shots if needed.



When requesting help, it is important that you run the Grub2Win diagnostics:


Create the diagnostic data by clicking the Diagnostics button on the Main Configuration screen.


You should then complete this diagnostics menu and click OK..






File C:\grub2\diagnose.7z will be generated and compressed.


You will then be given a choice:


Upload the diagnostic file to the Grub2Win support server


               or


Email the diagnostic file to me.

Just attach the C:\grub2\diagnose.7z file to the email.


The support email address is  drummerdp@users.sourceforge.net



Note:        Grub2Win uses FTP to upload diagnostic files to the support server.

 Your firewall software must be set to allow FTP port 21 traffic from grub2win.exe




Diagnostic items collected include the Grub2Win logs, partition lists, configuration files and BCD listings.





You can also send me a message directly at https://sourceforge.net/u/drummerdp/profile/ 


A free SourceForge account is required. You must be logged in to send messages.





Your comments and requests are welcome.


        - drummerdp
