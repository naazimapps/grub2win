GNU Grub Version 2.12 rc1




The Grub2Win setup installs the GNU Grub kernel beta version 2.12 rc1.


Although this is technically a beta release, it has been extensively tested and appears stable.

Ubuntu version 23.10 has upgraded to GNU Grub 2.12 rc1 as production.

OpenSuse has also upgraded to GNU Grub 2.12 rc1 as production.


If you wish to return to the older GNU Grub version 2.06, follow the instructions below:


1. Download the Prev206.zip file here:


       https://sourceforge.net/projects/sourcecode.grub2win.p/files/Prev206.zip/download



2. Unzip the downloaded Prev206.zip file and run InstallPrev206.exe. This will require administrator access.



3. The install program will update several files in your Grub2Win directory (usually C:\grub2).

    For EFI systems, it will also update the GNU Grub kernel files in your EFI partition.



4. Reboot your system. When the Grub2Win boot menu appears, press the  c  key on your keyboard.

    This will cause a GNU Grub command prompt to appear. Verify that the GNU Grub version is now 2.06.



5. Press the  esc  key on your keyboard to return to the Grub2Win boot menu.

    You can now use Grub2Win with the GNU Grub 2.06 code.
