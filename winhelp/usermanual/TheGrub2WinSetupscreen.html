Click here for the Latest Changes



Download the grub2win.zip file from SourceForge and unzip it to a location you choose.


Run the unzipped G2WInstall.exe program. The program requires administrator access.

It will download and extract the latest Grub2Win modules. The setup screen will now appear.



Grub2Win Setup






Select the disk where you want to set up the Grub2Win base directory, \grub2 (usually the C: drive).


Note: If there is already a \grub2 directory on the drive you select, it will be renamed to \grub2.old before the new directory is created.





Select The Target Drive

Select the drive where you wish to install the base \grub2 directory will be installed (usually C:). This selection will only appear if you have multiple NTFS partitions.


If there is already a \grub2 directory on the drive, a dialog will appear informing you that the existing directory will be renamed to \grub2.old. Click Confirm to agree and continue the installation.




Grub2Win desktop shortcut

Check this box if you would like to have a shortcut to Grub2Win placed on your desktop.





Run Grub2Win after the setup finishes

Check this box if you would like to run Grub2Win.






Setup

Click this when you are ready to start the installation.



A new \grub2 directory will be created on the drive you select. Your previous settings and backups will be migrated from \grub2.old if it exists so they will be available on the new \grub2 installation.


A progress box will appear showing the install steps as they occur.
