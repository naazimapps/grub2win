Uninstalling Grub2Win




Grub2Win is uninstalled using the standard Windows uninstall facilities.





For Windows 11, 10, 8, 7 and Vista systems:


Control Panel > Programs and Features


Click the Grub2Win entry to uninstall.





For XP systems:


Settings > Control Panel > Add or Remove Programs


Click the Grub2Win entry to uninstall.





If the standard Windows uninstall is not available, run this script instead:


C:\grub2\winsource\xxuninstall.bat






Notes:


On EFI systems, uninstall removes the Grub2Win entries from your EFI partition.


The uninstall attempts to restore the EFI default boot to its state before Grub2Win was installed.

If the previous EFI boot module is not found. The EFI default is set to boot Windows.


Uninstall also removes the Grub2Win desktop shortcut.


You may have to manually delete the empty C:\grub2 directory after the uninstall process completes.
